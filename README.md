# Personalized Weather Dashboard

A Flutter-based Android application that fetches real-time weather data from the Open-Meteo API using a student's index number. The app displays weather information in a user-friendly interface, handles errors gracefully, and supports offline access via local caching.

## Features

- Retrieve weather data based on student index number
- Real-time weather information display
- Offline access using local caching
- Error handling for API requests

## Tech Stack

- Flutter
- Dart
- Open-Meteo REST API
